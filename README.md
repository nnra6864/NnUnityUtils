# Nn Unity Utils

A collection of utilities and modules I use in most of my projects

## Dependencies

This package depends on:
1. [Nn Unity Easings](/nnra6864/NnUnityEasings)
2. [Nn Unity Text Utils](/nnra6864/NnUnityTextUtils)

## Installation

*This guide assumes the following structure*
```text
Project
├─> Modules
│   └─> NnUnityUtils
└─> ProjectName
    ├─> Assets
    └─> Packages
        └─> manifest.json
```

1. Navigate to your git repo root and run
    ```sh
    git submodule add ../../nnra6864/NnUnityUtils Modules/NnUnityUtils
    git submodule add ../../nnra6864/NnUnityEasings Modules/NnUnityEasings
    git submodule add ../../nnra6864/NnUnityTextUtils Modules/NnUnityTextUtils
    git submodule update --init --recursive
    ```

2. Add the following to `Packages/manifest.json` dependencies section
    ```json
    "com.nnra.nn-unity-utils": "file:../../Modules/NnUnityUtils",
    "com.nnra.nn-unity-easings": "file:../../Modules/NnUnityEasings",
    "com.nnra.nn-unity-text-utils": "file:../../Modules/NnUnityTextUtils",
    ```
